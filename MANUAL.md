## 1. Εισαγωγή

Το **Ergon Work Management** είναι εφαρμογή διαχείρισης εργασιών για επαγγελματικές ομάδες. Επιτρέπει την οργάνωση projects, την καταγραφή φωτογραφιών/βίντεο/μετρήσεων με GPS, τον συγχρονισμό δεδομένων και την παρακολούθηση της προόδου σε πραγματικό χρόνο.

**Για ποιους είναι:** Κατασκευαστικές εταιρείες, τεχνικές ομάδες, επιχειρήσεις που χρειάζονται τεκμηρίωση εργασιών στο πεδίο.

**Βασικοί ρόλοι:**
| Ρόλος | Περιγραφή |
|---|---|
| **Founder** (Ιδρυτής) | Πλήρης έλεγχος ομάδας |
| **Admin** | Διαχείριση μελών & projects |
| **Supervisor** | Επίβλεψη projects, δημιουργία projects |
| **Μέλος** | Εκτέλεση tasks |

**Διαθέσιμο:**
- 📱 **Mobile app** (Android)
- 🌐 **Web app**: [ergon-work.web.app](https://ergon-work.web.app)

## 4. Αίτηση Ένταξης σε Ομάδα

Αν δεν ανήκεις ακόμα σε ομάδα, εμφανίζεται αυτόματα η σελίδα αίτησης.

<img src="assets/ΑναμονηΓιαΕγκριση.png" width="320">

### Διαδικασία
1. Βλέπεις τα στοιχεία της ομάδας (όνομα, τύπος, αριθμός μελών)
2. Πάτα **"Αίτηση Σύνδεσης"**
3. Η αίτησή σου στέλνεται στον Admin/Founder της ομάδας
4. Περιμένεις έγκριση

### Καταστάσεις Αίτησης
| Κατάσταση | Τι σημαίνει |
|---|---|
| **Αναμονή έγκρισης** | Η αίτηση εστάλη, περιμένει Admin |
| **Απορρίφθηκε** | Ο Admin απέρριψε — μπορείς να ξαναστείλεις |

> Μόλις εγκριθεί η αίτησή σου, μεταφέρεσαι αυτόματα στις Ομάδες σου.

> **Περιορισμοί:**
> - Δεν μπορείς να τροποποιήσεις τον εαυτό σου
> - Admin δεν μπορεί να τροποποιήσει άλλον Admin ή Founder

### Αιτήματα Ένταξης *(Founder/Admin μόνο)*

<img src="assets/ΕγκρισηΚαιΕπιλωγηΡολουΓιαΝεαΜελη.png" width="320">

## 9. Σελίδα Υποέργου (Tasks)

Εδώ βλέπεις και διαχειρίζεσαι όλα τα tasks ενός Υποέργου.

<img src="assets/TaskPage.jpg" width="250">

### Header
- **Τίτλος Υποέργου**
- **Progress** (π.χ. "3/7 Ολοκληρώθηκαν")
- Κουμπιά πάνω δεξιά:

| Εικονίδιο | Λειτουργία |
|---|---|
| ☁️ (σύννεφο) | Manual sync — εμφανίζεται μόνο αν υπάρχουν εκκρεμή |
| 🕐 (ρολόι) | Ιστορικό δραστηριοτήτων |
| 🖨️ (εκτυπωτής) | PDF Export |

### Λίστα Tasks
Κάθε task εμφανίζεται ως κάρτα με:
- **Εικονίδιο τύπου** (📷 φωτογραφία / 🎬 βίντεο / 🔧 μέτρηση / 📄 κείμενο)
- **Τίτλο** και περιγραφή
- **Κατάσταση**: Πράσινο = Ολοκληρωμένο, Μπλε = Εκκρεμές
- **Thumbnail** για φωτογραφίες/βίντεο με αριθμό (badge)
- **Αξία** για μετρήσεις/κείμενο (αν είναι ολοκληρωμένο)
- **"Αναμονή Δικτύου"** (πορτοκαλί badge) → αποθηκεύτηκε offline

**Tap σε task** → Ανοίγει το task για εκτέλεση (βλ. §10).
*(Αν το Υποέργο είναι κλειστό, το tap σε μέτρηση/κείμενο δεν κάνει τίποτα.)*

> ⚠️ **ΚΡΥΦΗ ΛΕΙΤΟΥΡΓΙΑ: Long press σε task card** (~0.5 δευτ.)
> Ανοίγει μενού: **Επεξεργασία** / **Διαγραφή**
> *(Απενεργοποιημένο αν το Υποέργο είναι κλειστό.)*

### FAB (+) — Δημιουργία Task
Κουμπί κάτω δεξιά (μπλε με +).

<img src="assets/CreateNewTask.jpg" width="250">

1. Πάτα το **+**
2. Επέλεξε τύπο: Φωτογραφία / Βίντεο / Μέτρηση / Κείμενο
3. Συμπλήρωσε τίτλο (υποχρεωτικό) και περιγραφή (προαιρετικό)
4. Πάτα "Αποθήκευση"

> **Σημαντικό:** Το FAB εμφανίζεται **μόνο** όταν το Υποέργο είναι **ανοιχτό**. Το κλείσιμο γίνεται αποκλειστικά από το Admin Panel.

### Κλειστό Υποέργο 🔒

Όταν ένα Υποέργο κλειστεί από τον Admin:

- Εμφανίζεται **banner** κάτω: *"Αυτό το υποέργο είναι κλειστό"* 🔒
- **Δεν επιτρέπεται καμία αλλαγή:**
  - ❌ Προσθήκη νέων tasks (FAB κρυφό)
  - ❌ Επεξεργασία / Διαγραφή tasks (long press απενεργοποιημένο)
  - ❌ Εισαγωγή / Αλλαγή μέτρησης ή κειμένου
  - ❌ Ανέβασμα φωτογραφίας / βίντεο (κουμπί ADD κρυφό)
  - ❌ Διαγραφή φωτογραφίας / βίντεο (κουμπί trash κρυφό)
  - ❌ Επανεπεξεργασία φωτογραφίας (κουμπί brush κρυφό)
- Μπορείς μόνο να **βλέπεις** το περιεχόμενο και να **εξάγεις PDF**

### Κατάσταση (Status) Υποέργου

| Κατάσταση | Πότε εμφανίζεται |
|---|---|
| **Ενεργό** | Δεν έχει ολοκληρωθεί κανένα task |
| **Εκκρεμές** | Μερικά tasks ολοκληρωμένα |
| **Ολοκληρωμένο** | Μόνο όταν το κλείσει ο Admin (🔒 Κλείσιμο) |

> Το status **ΔΕΝ** γίνεται "Ολοκληρωμένο" αυτόματα όταν τελειώσουν όλα τα tasks. Απαιτείται ρητό κλείσιμο από Admin.

## 10. Τύποι Tasks

### 📷 Task Φωτογραφίας

**Tap στο task:**
1. Επέλεξε **Κάμερα** ή **Γκαλερί**
2. **Αν Κάμερα:** Τραβάς φωτογραφία → ανοίγει αυτόματα ο **Editor Σχεδίασης**

**Editor Σχεδίασης:**
- **Σχεδίασε** πάνω στη φωτογραφία (annotations, σημάνσεις, κύκλοι κλπ)
- **6 χρώματα**: 🔴 κόκκινο, 🟡 κίτρινο, 🟢 πράσινο, 🔵 μπλε, ⬜ λευκό, ⬛ μαύρο
- **3 μεγέθη πινέλου** (λεπτό / μεσαίο / χοντρό)
- **Pan**: Σύρε με 1 δάχτυλο για μετακίνηση της εικόνας
- **↩ Undo**: Αναίρεση τελευταίας γραμμής
- **🔄 Reset**: Διαγραφή όλων των σχεδίων
- Πάτα **Αποθήκευση** για να κρατήσεις την εικόνα

**Αυτόματα:**
- Καταγράφεται **GPS τοποθεσία** και αποθηκεύεται στο EXIF
- Κάθε task μπορεί να έχει **πολλές φωτογραφίες** (badge με αριθμό)

Πάτα οποιαδήποτε φωτογραφία στη γκαλερί για να ανοίξει μενού επιλογών:
- **Εμφάνιση τοποθεσίας** → Ανοίγει Google Maps με ακριβείς συντεταγμένες
- **Κοινοποίηση** → Μοιράζεσαι τη φωτογραφία
- **Επανεπεξεργασία** → Ξανανοίγει τον editor *(μόνο αν ανοιχτό Υποέργο)*
- **Διαγραφή** → Διαγράφει τη φωτογραφία *(μόνο αν ανοιχτό Υποέργο)*

---

### 🎬 Task Βίντεο

**Tap στο task:**
1. Επέλεξε **Κάμερα** ή **Γκαλερί**
2. Το βίντεο **συμπιέζεται αυτόματα** και ενσωματώνεται η GPS τοποθεσία
3. Κάθε task μπορεί να έχει **πολλά βίντεο**

Πάτα οποιοδήποτε βίντεο στη γκαλερί για να ανοίξει μενού επιλογών:
- **Εμφάνιση τοποθεσίας** → Google Maps
- **Διαγραφή** → με επιβεβαίωση *(μόνο αν ανοιχτό Υποέργο)*

---

### 📏 Task Μέτρησης

**Tap στο task** *(μόνο αν ανοιχτό Υποέργο)*:
1. Ανοίγει παράθυρο εισαγωγής τιμής
2. Γράψε την τιμή (αριθμό ή κείμενο, π.χ. "2.45m", "OK", "Αποτυχία")
3. Πάτα **Αποθήκευση** → task γίνεται "Ολοκληρωμένο", η τιμή εμφανίζεται στην κάρτα

**Διαγραφή τιμής:**
- Πάτα το εικονίδιο 🗑️ (καλάθι) μέσα στο παράθυρο
- Εμφανίζεται επιβεβαίωση "Είσαι σίγουρος;"
- Αν επιβεβαιώσεις, η τιμή διαγράφεται και το task επιστρέφει σε "Εκκρεμές"

---

### 📝 Task Κειμένου (General)

**Tap στο task** *(μόνο αν ανοιχτό Υποέργο)*:
1. Ανοίγει παράθυρο εισαγωγής ελεύθερου κειμένου (πολλές γραμμές)
2. Γράψε σημείωση, παρατήρηση, κλπ
3. Πάτα **Αποθήκευση** → task γίνεται "Ολοκληρωμένο"

**Διαγραφή κειμένου:**
- Όπως στη Μέτρηση — το 🗑️ ζητά επιβεβαίωση

## 11. Ιστορικό Δραστηριοτήτων

Από τη σελίδα Υποέργου, πάτα το εικονίδιο **🕐 (ρολόι)** πάνω δεξιά.

Βλέπεις τις **τελευταίες 50 ενέργειες** σε πραγματικό χρόνο, με:
- Όνομα χρήστη που έκανε την ενέργεια
- Τι έκανε (π.χ. "Πρόσθεσε φωτογραφία στο «Βάση Α»")
- Πότε (π.χ. "πριν 5 λεπτά", "χθες")

**Καταγράφονται:**
- Προσθήκη / Διαγραφή task
- Αποστολή / Διαγραφή / Επανεπεξεργασία φωτογραφίας ή βίντεο
- Καταχώριση / Διαγραφή τιμής μέτρησης ή κειμένου
- Ολοκλήρωση task

## 12. PDF Export

Από τη σελίδα Υποέργου, πάτα το εικονίδιο **🖨️ (εκτυπωτής)**.

Δημιουργείται αυτόματα αναφορά που περιλαμβάνει:
- Τίτλο Υποέργου και ημερομηνία εξαγωγής
- Όλα τα tasks με τύπο, κατάσταση και ημ/νία ολοκλήρωσης
- Φωτογραφίες σε πλήρη ανάλυση
- Τιμές μετρήσεων και κείμενα
- Συνδέσμους GPS (κλικάρισμα → Google Maps)

Ανοίγει το **dialog εκτύπωσης** — μπορείς να αποθηκεύσεις ως PDF ή να εκτυπώσεις.

## 13. Offline Λειτουργία

Η εφαρμογή **λειτουργεί και χωρίς internet**. Δεν χρειάζεται να ανησυχείς αν χάσεις σύνδεση στο πεδίο.

### Τι μπορείς να κάνεις offline:
- Δεις τη λίστα ομάδων και projects (από cache)
- Δημιουργήσεις νέα tasks
- Τραβήξεις φωτογραφίες και βίντεο
- Καταχωρήσεις μετρήσεις και κείμενα

### Τι γίνεται με τα δεδομένα offline:
- Αποθηκεύονται **τοπικά** στη συσκευή
- Εμφανίζουν badge **"Αναμονή Δικτύου"** (πορτοκαλί)
- **Ανεβαίνουν αυτόματα** μόλις βρεθείς σε σύνδεση

### Ρυθμίσεις Sync:
| Ρύθμιση | Πού | Τι κάνει |
|---|---|---|
| Αυτόματος (WiFi) | — | Προεπιλογή mobile, ανεβάζει μόνο σε WiFi |
| Χειροκίνητος | "Ομάδες μου" → κάρτα sync | Ανεβάζει τώρα |

> **Web app:** Αν υπάρχει σύνδεση internet, ο sync γίνεται αμέσως χωρίς διάκριση WiFi/cellular.

## 14. Google Drive Sync

Δυνατότητα **αυτόματου backup** όλων των αρχείων στο Google Drive σου.

### Ρύθμιση
1. Σελίδα Ομάδας → ⋮ → **Google Drive Sync**
2. Πάτα **"Σύνδεση Google Drive"**
3. Σύνδεση με Google λογαριασμό
4. Ολοκλήρωση — εμφανίζεται το email σύνδεσης

> Απαιτείται σύνδεση στο internet. Αν είσαι offline, εμφανίζεται μήνυμα "Offline".

### Λειτουργία
- **Αυτόματος sync**: 30 δευτ. μετά από κάθε αλλαγή (μόνο WiFi)
- **Χειροκίνητος sync**: Κουμπί "Συγχρονισμός Τώρα"
- **Άνοιγμα Drive**: Κουμπί "Άνοιγμα στο Drive" → ανοίγει τον φάκελο στο browser

### Δομή Φακέλων στο Drive
```
Ergon Work Management/
└── [Όνομα Ομάδας]/
    └── [Όνομα Project]/
        └── [Όνομα Υποέργου]/
            ├── Φωτογραφίες/
            ├── Βίντεο/
            ├── Μετρήσεις/
            └── Κείμενο/
```

### Αποσύνδεση
Κουμπί "Αποσύνδεση Google Drive" — ζητά επιβεβαίωση.

## 15. Admin Panel (Web)

Ένα **browser-based εργαλείο** για Founders και Admins, για διαχείριση από υπολογιστή ή κινητό browser.

### Πρόσβαση
- Σελίδα Ομάδας → ⋮ → **Admin Panel**
- Ανοίγει αυτόματα στο browser: `ergon-work.web.app/admin`
- Απαιτεί internet και σύνδεση *(μόνο Founder/Admin)*

### Tab: Μέλη

<img src="assets/AdminPanelMembers.jpg" width="280">

- Λίστα όλων των μελών με ρόλο
- **Αλλαγή ρόλου**: dropdown menu ανά μέλος
- **Αφαίρεση μέλους**: κουμπί "Kick" (με επιβεβαίωση)

### Tab: Projects

<img src="assets/AdminPanelProgectPage.jpg" width="280"> <img src="assets/AdminPanelTaskPage.jpg" width="280">

- Λίστα Projects ανά group με status badge
- **Κλικ σε Υποέργο** → λεπτομέρειες tasks:
  - Κάθε task έχει checkbox για **πολλαπλή επιλογή**
  - "Επιλογή όλων" checkbox στην κορυφή
  - Κουμπί **"Διαγραφή επιλεγμένων (N)"** → διαγράφει τα tasks **και** τα αρχεία τους από το Storage αυτόματα
  - Κλικ σε μεμονωμένο task → επεξεργασία (τίτλος, κατάσταση, τιμή)
- **🔒 Κλείσιμο / 🔓 Άνοιγμα Υποέργου**: Κουμπί στο header κάθε Υποέργου
  - **Κλείσιμο** → status = "Ολοκληρωμένο", κλειδώνει όλες τις αλλαγές στην app
  - **Άνοιγμα** → status = "Ενεργό", ξεκλειδώνει τις αλλαγές
- **Διαγραφή Υποέργου** (κόκκινο κουμπί, με επιβεβαίωση) — διαγράφει και τα media από Storage

### Tab: Storage

<img src="assets/AdminPanelStoragePage.jpg" width="280">

- Λίστα αρχείων ανά project (φωτογραφίες 🖼️, βίντεο 🎬)
- Κλικ σε project για ανάπτυξη αρχείων
- Κάθε αρχείο έχει **checkbox** για πολλαπλή επιλογή
- "Επιλογή όλων" per project
- **"Διαγραφή επιλεγμένων (N)"** για μαζική διαγραφή αρχείων από Storage
- Μεμονωμένο κουμπί "Διαγραφή" ανά αρχείο

## 18. Συχνές Ερωτήσεις & Tips

| Πρόβλημα / Ερώτηση | Απάντηση |
|---|---|
| "Πώς μπαίνω στην ομάδα;" | Άνοιξε την εφαρμογή → αίτηση ένταξης → περίμενε έγκριση από Admin |
| "Δεν βλέπω το υποέργο μου" | Άνοιξε το Project → ελέγξε το φίλτρο status — μπορεί να φιλτράρεις μόνο "Ενεργά" |
| "Το task δεν ανέβηκε" | Βλέπε το 🟠 badge "Αναμονή Δικτύου" — κάνε manual sync |
| "Πού τραβήχτηκε η φωτογραφία;" | Tap στη φωτογραφία → "Εμφάνιση τοποθεσίας" |
| "Πώς διαγράφω μια μέτρηση;" | Tap στο task → πάτα 🗑️ → επιβεβαίωση |
| "Πώς επεξεργάζομαι ένα task;" | Long press στην κάρτα του task → "Επεξεργασία" |
| "Πώς μεταφέρω υποέργο σε άλλο project;" | Long press στο υποέργο → "Μεταφορά σε άλλο Project" |
| "Πώς διαγράφω ένα project;" | Πάτα το 🗑️ δίπλα στον τίτλο του project (μόνο αν είναι άδειο) |
| "Γιατί δεν βλέπω το FAB (+);" | Το Υποέργο είναι **κλειστό** από τον Admin — ζήτα άνοιγμα από Admin Panel |
| "Πώς κλείνω ένα Υποέργο;" | Μόνο από Admin Panel → Projects → επίλεξε Υποέργο → 🔒 Κλείσιμο |
| "Το status δεν γίνεται Ολοκληρωμένο αυτόματα;" | Σωστό — απαιτείται ρητό κλείσιμο από Admin Panel |
| "Το Project έγινε πράσινο — τι σημαίνει;" | Όλα τα Υποέργα του είναι κλειστά — Project ολοκληρωμένο ✅ |
| "Πώς επανεπεξεργάζομαι φωτογραφία;" | Ανοίξε τη γκαλερί → tap στη φωτογραφία → "Επανεπεξεργασία" *(μόνο ανοιχτό Υποέργο)* |
| "Πώς κάνω manual sync;" | Σελίδα "Ομάδες μου" → tap στην κάρτα sync (πορτοκαλί) |
| "Η σύνδεση Drive δεν λειτουργεί offline" | Απαιτεί internet — δες μήνυμα "Offline" |
| "Δεν βλέπω το καμπανάκι αιτημάτων" | Εμφανίζεται μόνο σε Founder/Admin |
